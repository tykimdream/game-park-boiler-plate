# 세션 핸드오프 규칙

## 세션 시작
- HANDOFF.md가 존재하면 읽고 이전 컨텍스트를 파악한다
- 이전 세션의 다음 단계와 차단 요소를 확인한다

## 세션 종료
- 사용자가 종료 신호("끝", "마무리", "종료", "오늘은 여기까지", "다음에 이어서")를 보내면 HANDOFF.md를 자동 갱신한다
- 갱신 항목: 마지막 업데이트, 현재 상태, 다음 단계, 중요 컨텍스트
- 갱신 후 "HANDOFF.md를 업데이트했습니다"라고 알린다

## 수동 저장
- `/handoff` 커맨드로 언제든 수동 저장 가능
- `/resume` 커맨드로 이전 세션 복원 가능
