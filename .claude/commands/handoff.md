현재 세션의 작업 컨텍스트를 HANDOFF.md에 저장한다.

## 절차

1. `git status`와 `git branch`를 확인하여 현재 상태를 파악한다
2. 이번 세션에서 수행한 작업, 결정 사항, 진행 상황을 정리한다
3. HANDOFF.md의 모든 섹션을 갱신한다:
   - **마지막 업데이트**: 현재 일시 + 한줄 요약
   - **현재 상태**: 진행 중/완료 체크리스트
   - **결정 사항**: 이번 세션의 결정 추가
   - **다음 단계**: 우선순위 순 정리
   - **차단 요소**: 미해결 이슈 기록
   - **중요 컨텍스트**: 브랜치, 워크플로우 단계, 관련 문서, 기술 노트
4. 저장 완료 후 요약을 출력한다

$ARGUMENTS가 있으면 세션 요약에 포함한다: $ARGUMENTS
